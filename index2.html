Apuntes de la clase

Establece el tipo de la caja que genera cada elemento.
Cada elemento HTML tiene un valor de visualizacion predeterminado segun el tipo de elemento que sea.
El valor de visualizacion para la mayoria es block e inline.

display-block: Siempre comienza una nueva linea y ocupa todo el ancho disponible.

display-inline: No comienza en una nueva linea y solo ocupa el ancho que sea necesario, no acepta el width y height.

display-inline-block: Permite establecer un ancho y una altura en el elemento, ademas se represtan los margin y padding que con display-inline no son respetados.

display-none: Se usa comunmente con JavaScript para ocultar y mostrar elemento cuando el usuario realiza alguna accion como pulsar un boton o un enlace sin eliminarlos ni crearlos nuevamente.

display-table: Permiten asignar un comportamiento de tabla a los elementos que haya adentro aninados a la etiqueta donde estas colocando el display-table. Se pueden usar otra serie de valores de display que corresponde con los valores de distancia.
