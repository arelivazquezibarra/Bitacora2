Apuntes de la clase

En un intento por reducir el uso del codigo Java y para estandalizar funciones populares CSS3 no solo colore diseño y estilo si no tambien forma y movimiento.

Este nivel de cambio convierte CSS3 en una tecnologia practicamente inedita comparada con versiones anteriores.

Border-radius: Esta propiedad genera esquinas redondeadas para la caja formada por el elemento.

@font-face: Esta regla permite cargar y usar cualquier fuente que necesitemos, luego de esto podemos asignar la fuente a cualquier elemento del documento.

Linear-gradient: Esta funcion puede ser aplicada a las propiedades background o background-imagen para generar un gradiante lineal.

Radial-gradient: Esta funcion puede ser aplicada a las propedades background o background-imagen para generar un grandiante radial.

RGBA: Esta funcion es una mejora del rgb con un canal alfa que especifica la opacidad de un color.

HSLA: HSL significa tono, saturacion y luminosidad. Los valores de color HSLA son una extencion de los valores de color HSL con un canal alfa, que especifica la opacidad para un color.

Outline: Esta propiedad fue mejorada con la incorporacion de otra propiedad llamada outline-offset. Ambas propiedades combinadas generan un sengundo borde alejado del borde original del elemento.

Border-imagen: Esta propiedad crea un borde con una imagen personalizada. Necesita que border sea declarado previamente y toma al menos tres parametros URL, Tamaño, palabra clave.

Transform: Esta propiedad modifica la forma de un elemento.

Resumen


Las nuevas propiedades de css3 nos han facilitado las cosas al momento de crear hojas de estilo, teniendo asi mismo una claridad de lo que est haciendo e implementado, ayudando a la estetica de nuestros proyectos.
