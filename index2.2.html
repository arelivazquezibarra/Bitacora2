Apuntes de la clase

Atributo id: El id es como un nombre que identifica un elemento especifico en html, para seleccionar un elemento con un id especifico, se escribe un caracter numeral (#).

Atributo class: Este atributo es mas flexible y puede ser asignado a varios elementos en el html que compraten un dise√±o similar.

Elementos: Un selector de elementos en css es una regla que selecciona todos los elementos de un tipo especifico en el documento html. Es el selector mas basico y se refiere directamente al nombre de uns etiqueta html.

Pseudoclases:
nth-child: Nos permite encontrar un hijo especifico.
nth-child(odd) :nth-child(even): Las palabras clave odd y even se utilizan para selecionar elementos con base en su posicion en la secuencia de hijos.
Firts-child: Referencia solo al primer hijo
Last-child: Referencia solo el ultimo hijo.
Only-child: Afecta un elemento siempre y cuando sea el unico hijo disponible.
:not(): Nos permite declarar una excepcion.
Selector de atributo: selecciona elementos html en funcion de la presencia o el valor de uno de sus atributos.
-[atributo^="inicio"]: Seleciona elementos cuyo atributo comienza con un valor especifico.
-[atributo$="final"]: Seleciona elementos cuyo atributo termina con un valor especifico.
-[atributo*="cadena"]: Seleciona elementos cuyo atributo contiene una cadena especifica en cualquier posicion.

Selectores de relacion: Permite seleccionar elementos basados en su relacion con otros elementos dentro del documento. Estos selectores son utiles para aplicar estilos a elementos.
Hijos directos: Seleciona los elementos que son hijos inmediatos de un elemento padre.
Hermanos adyacentes: Selecciona un elemento que es hermano inmediato, es decir, esta justo despues de otro elemento.
Hermanos generales: Seleciona todos los elementos que son hermanos de un elemento anterior , es decir, esta antes, sin importar si son hermanos inmediatos o no.
