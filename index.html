Propiedad position

Ideas principales


Hay cinco valores de posicion diferente:

  • Position Static
  • Position Relative
  • Position Absolute
  • Position Fixed
  • Position Sticky


Overflow:hidden :Hace que la posicion sticky no trabaje/funcione.

Sticky no va a funcionar cuando el elemento contenedor no tiene una altura, la barra de navegacion es aparte/afuera.

Apuntes de la clase


La propiedad de posicion especifica el tipo de metodo de posicionamiento utilizado para un elemento.
Los elementos se posicionan usanso las propiedades top, bottom, left y right.
Tambien funcionan de manera diferente dependiendo del valor de propiedad.

Position static: Los elementos posicionados estaticos no se ven afectados con los elementos top, bottom, left y right, un elemento con position static no esta posicionado de ninguna manera especial, siempre se posiciona de acuerdo al flujo normal de la pagina.
Position Relative: Desplaza un elemento de su posicion original establecida mediante el posicionamiento normal.
Position Absolute: Depende del posicionamiento del elemento contenedor.
Position fixed: Las propiedades se utilizan para posicionar el elemento.
Position sticky: Alterna entre relativo y fijo, dependiendo de la posicion de desplazamiento.

Resumen


En general sabemos que la propiedad position sirve para el acomodo de elementos contenedor dandole a cada elemento un espacion en el cual estar y que no se vea nuestro sitio tan amontonado o que no haya un elemento encima del otro.
